# PHP_Shoping

## Mô tả chung

**PHP_Shoping** là dự án ứng dụng web thương mại điện tử cơ bản, cho phép người dùng duyệt sản phẩm, thêm vào giỏ hàng, đặt hàng và quản lý sản phẩm / đơn hàng từ phía quản trị. Mục tiêu là xây dựng hệ thống mua bán nhỏ với các chức năng thiết yếu.

---

## Tính năng chính

- Hiển thị danh sách sản phẩm theo danh mục  
- Xem chi tiết sản phẩm  
- Thêm sản phẩm vào giỏ hàng, chỉnh sửa số lượng, xoá sản phẩm  
- Thanh toán / đặt hàng  
- Quản lý đơn hàng (xem, xử lý)  
- Quản lý sản phẩm (thêm, sửa, xoá)  
- Quản lý người dùng (đăng ký, đăng nhập, phân quyền)  
- Giao diện thân thiện với người dùng & quản trị  

---

## Công nghệ sử dụng

-**Ngôn ngữ**: PHP
- **Cơ sở dữ liệu**: MySQL 
- **Thư viện / phụ trợ**: Bootstrap5
